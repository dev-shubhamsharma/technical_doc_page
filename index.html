Animated Preloader

https://codepen.io/shu217/pen/VwYpOyj

A webpage with high quality of images takes some time to load and makes web pages ready for complete process. A preloader prevents the user to click on any content before the loading has finished.

When page is ready, preloader automatically gets hide and thus user can continue..

html

<body>
  <div class="loader">
    <div class="content">
      <div class="circle"></div>
      <div class="circle"></div>
      <div class="circle"></div>
      <div class="circle"></div>
    </div>
  </div>
</body>

*
{
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

{
  width: 100vw;
  height: 100vh;
  background:#f1c40f;
  display: flex;
  justify-content: center;
  align-items: center;
}

{
  width: 8vmin;
  height: 8vmin;
  border-radius: 50%;
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
  animation: animate-content 4s linear infinite;
}

{
  width: 50%;
  height: 50%;
  border-radius: 50%;
}

.circle:nth-child(1)
{
  background: #e74c3c;

}

.circle:nth-child(2)
{
  background: #27ae60;

}

{

Button Hover Effects

https://codepen.io/shu217/pen/YzPrzVq?editors=1000

Html default buttons are simple. to make the buttons impressive we can use the 'css'. here is the some examples..

Here i have created Four buttons for four direction effects individually.

  • from top
  • from right
  • from bottom
  • from left

html

<body>
  <div class="container">
    <button id="from-top">Hover me<span></span></button>
    <button id="from-right">Hover me<span></span></button>
    <button id="from-bottom">Hover me<span></span></button>
    <button id="from-left">Hover me<span></span></button>
  </div>
</body>

css

:root
{
   --c: goldenrod;
}

body
{
   min-height: 100vh;
   background: #000;
   display: flex;
   align-items: center;
   justify-content: center;
}

.container
{
   box-sizing: content-box;
   display:flex;
   flex-direction: column;
   align-items: center;
}

button
{
   width: 40vmin;
   height: 9vmin;
   border: 0.5vmin solid var(--c);
   margin: 4vmin;
   background: #000;
   color: var(--c);
   font-size: 3vmin;
   text-transform: uppercase;
   letter-spacing: 0.4px;
   position: relative;
   z-index: 1;
   overflow: hidden;
   transition: 0.3s ease-in;
   outline: none;
}

button:hover
{
   color: #000;
}

button span
{
   width: 100%;
   height: 100%;
   background: var(--c);
   position: absolute;
   z-index: -1;
   transition: 0.5s ease-in;
}

#from-top span
{
   top: -100%;
   left: 0;
}

#from-top:hover span
{
   top: 0;
}

#from-bottom span
{
   left: 0;
   bottom: -100%;
}

#from-bottom:hover span
{
   bottom: 0;
}

#from-left span
{
   left: -100%;
   top: 0;
}

#from-left:hover span
{
   left: 0;
}

#from-right span
{
   right: -100%;
   top: 0;
}

#from-right:hover span
{
   right: 0;
   top: 0;
}
